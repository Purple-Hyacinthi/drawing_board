# Drawing Board Pro 用户指南

## 1. 产品定位

当前版本前端为**纯画板模式**：

- 打开应用即进入画板
- 界面不包含首页、画廊和多余导航
- 面向中文用户，页面文案默认中文

## 2. 快速开始

### 2.1 启动前端

```bash
cd frontend
npm install
npm run dev
```

浏览器访问：`http://127.0.0.1:3000`

### 2.2 进入画板

- 访问根路径 `/` 即为画板页
- 任意未知路径会自动跳转到画板页

## 3. 画板使用

### 3.1 绘制

- 鼠标左键按下后拖动即可绘制
- 松开左键结束当前笔触
- 默认是深色基础画笔

### 3.2 压感输入

- 使用压感笔时会读取设备压力
- 鼠标输入会使用默认力度
- 压感会影响线条粗细与笔触表现

### 3.3 画布说明

- 画布区域默认较大，支持滚动查看
- 移动端和小屏设备会自动收紧边距

## 4. 当前版本边界

- 本版本前端只保留画板核心能力
- 不提供首页、画廊、顶部导航、底部信息栏
- 页面内不提供复杂工具面板

如需图层、导出、历史等复杂流程，可通过后端 API 或后续版本界面扩展实现。

## 5. 常见问题

### 5.1 打不开页面

请确认前端开发服务已启动，并检查终端是否有端口占用错误。

### 5.2 画不出来

请确认在白色画布区域内按下鼠标左键后拖动。

### 5.3 页面文字不是中文

请确认浏览器未启用自动翻译插件或页面翻译扩展。
